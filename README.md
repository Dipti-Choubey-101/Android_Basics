A comprehensive collection of **Android development fundamentals** implemented using **Kotlin** and **Android Studio**. This repository contains beginner-friendly projects that demonstrate essential Android concepts through practical examples.

Whether you're a student, beginner, or aspiring Android developer, these examples will help you build a strong foundation in Android application development.

## 💻 Getting Started

### Clone the repository

```bash
git clone https://github.com/Diptanil-Sen/Android_Basics.git
```

### Open in Android Studio

1. Launch Android Studio
2. Select **Open an Existing Project**
3. Choose the cloned repository
4. Allow Gradle to sync
5. Run any module on an Emulator or Physical Device
